# 库区库位管理

## 业务定位

库区和库位构成仓库内部的二级空间结构。库区是仓库内的逻辑分区（如合格品区、不合格品区），库位是最小存储单元，直接承载库存商品。

## 核心概念

| 概念 | 说明 |
|------|------|
| 库区 (StoreArea) | 仓库内的逻辑分区，介于仓库和库位之间 |
| 库位 (StoreLocation) | 最小存储单元，库存实际存放位置 |
| 库位类型 | 合格库位 / 不合格库位 |
| 库位分类 | 通用 / 医疗器械 / 非医疗器械 |
| 跟台库位 | 专门用于跟台手术入库的库位 |
| 库位定位 | 通道号+位置类型+货架号+层号+列号，精确物理定位 |
| 校验类型 | 库位是否校验货品属性 |

## 业务规则

### 库区管理

**新增库区**：
- 同一仓库下库区名称不可重复
- **三方仓库限制**：
  - 库区名称必须为"合格库区"或"不合格库区"
  - 最多只能创建 2 个库区（1 个合格 + 1 个不合格）

**修改库区**：
- 库区名称在同一仓库下不可与其他库区重名（排除自身）
- 三方仓库同样受限于 2 个库区上限

**删除库区**：
- 库区下已创建库位时不允许删除
- 删除库区时级联删除关联的计量数据

### 库位管理

**新增库位**：
1. 操作受并发控制保护，同一仓库不能同时添加库位
2. **库位码唯一性**：同一仓库下库位码不可重复
3. **公司配置校验**：
   - 仓库下必须配置至少一个公司
   - 指定的公司必须在仓库配置范围内
   - 若仓库仅配置一个公司，自动填充该公司
4. **跟台库位必填项**：当库位为跟台库位时，手术类型和模板类型必填
5. **库位类型默认值**：未指定时默认为合格库位
6. **三方仓库限制**：
   - 库位所属库区必须是"合格库区"或"不合格库区"
   - 合格库区只能创建合格库位，不合格库区只能创建不合格库位
   - 每种类型只能创建 1 个库位
   - 库位分类只能选"通用"

**库位定位校验**：
- 定位信息包含：通道号、通道位置类型、货架号、层号、列号
- **启用库位推荐的仓库**：定位信息必须完整填写，否则报错
- **定位信息唯一**：填写完整的定位信息在同一仓库下必须唯一

**修改库位**：
- **不可修改字段**：库位码、库位类型、库位分类、跟台属性
- **跟台库位额外不可修改**：公司、手术类型、模板类型
- **有库存或有未完成出库单时的额外限制**：校验类型、公司、客户、供应商、业务单元、状态均不可修改

**删除库位**：
- 逐个检查库位是否有库存
- 有库存的库位不删除，记录为失败跳过
- 无库存的库位正常删除
- 不会因单个失败中断整体操作

**批量导入库位**：
- 支持 Excel 批量导入
- 默认状态为启用
- 在事务中批量写入

## 业务流程

### 库位新增完整流程

```
用户提交新增库位请求
  │
  ├── 锁定仓库，防止并发添加
  │
  ├── 校验库位码唯一性
  │
  ├── 校验公司配置（仓库是否绑定公司）
  │
  ├── 跟台库位 → 校验手术类型+模板类型必填
  │
  ├── 库位定位校验（通道/货架/层/列）
  │   ├── 启用库位推荐 → 定位信息必须完整
  │   └── 定位完整 → 仓库内唯一性校验
  │
  ├── 三方仓库 → 库区+库位类型匹配校验
  │
  └── 保存库位
```

### 库位删除流程

```
用户选择多个库位删除
  │
  ├── 逐个检查库位库存数量
  │   ├── 有库存 → 记入失败列表，跳过
  │   └── 无库存 → 记入可删除列表
  │
  ├── 批量删除无库存库位
  │
  └── 返回失败列表给用户
```

## 特殊场景处理

1. **库位码重复数据**：如果同一仓库下存在多条相同库位码的记录（数据异常），直接报错要求联系管理员
2. **库位与出库单关联**：库位修改校验不仅检查库存数量，还检查是否存在未完成的出库单

## 操作指引

### 库位列表

**查询条件**：公司、所属仓库（级联公司）、所属库区（级联仓库）、库位编号、校验类型、库位类型、库位分类、通道、通道位置、货架、层、列、备注、特殊条件、供应商、业务单元。

**级联过滤**：公司 → 仓库 → 库区。

**操作按钮**：
- **新增**：打开新建库位表单
- **编辑**：需选中一条记录
- **删除**：需选中至少一条记录，后端逐个检查库存后删除
- **打印条码**：需选中至少一条有库位码的记录
- **导入**：下载模板、上传 Excel、下载错误报告
- **导出数据**：异步导出，完成后站内消息通知

### 新建库位表单

**必填字段**：所属仓库、所属库区、库位码、库位类型、校验类型、状态、公司、跟台库位、库位分类。

**级联联动**：
1. 仓库变更时清空库区选择
2. 仓库变更时加载仓库关联的公司列表
3. 库位分类为"非医疗器械"时自动设置库位类型为"不合格"
4. 跟台库位时自动将校验类型设为"不校验"且不可修改，同时显示手术类型和跟台模板字段

**库位码格式**：大写字母开头 + 数字/大小写字母/连字符。

**库位推荐定位**：当仓库启用库位推荐时，显示通道/货架/通道位置/层/列 5 个定位字段，全部必填。

### 编辑库位

- 所属仓库、所属库区、库位码、库位分类、库位类型、跟台库位均不可修改
- 可修改：校验类型、供应商、业务单元、客商、公司、状态、定位信息、备注

### 库位导入

支持下载导入模板、上传 Excel 批量导入、下载错误报告。导入成功后自动刷新列表。

## 数据模型

**库区核心字段**：

- 库区 ID
- 库区名称
- 所属仓库

**库位核心字段**：

- 库位 ID
- 库位编码（仓库内唯一）
- 所属仓库
- 所属库区
- 所属公司
- 库位类型（合格 / 不合格）
- 库位分类（通用 / 医疗器械 / 非医疗器械）
- 是否跟台库位
- 手术类型（跟台库位专用）
- 模板类型（跟台库位专用）
- 校验类型
- 状态（启用 / 停用）
- 定位信息（通道号/位置类型/货架号/层号/列号）

## 外部系统依赖

| 外部系统/模块 | 用途 |
|---------------|------|
| 仓库管理 | 获取仓库类型、库位推荐配置 |
| 公司管理 | 校验公司是否在仓库配置范围内 |
| 库存查询 | 检查库位是否有库存 |
| 出库查询 | 检查库位是否有未完成出库单 |
| 计量管理 | 库区删除时级联删除计量数据 |
