# 自动单据人员配置

## 业务定位

自动单据人员配置用于设定收货、验收、出库等流程中自动填充的操作人员和复核人员。通过预配置人员角色，在自动生成单据时免去手动选择人员的步骤，提高效率并确保职责分离。

## 核心概念

| 概念 | 说明 |
|------|------|
| 自动单据人员配置 | 按公司+仓库维度配置的自动化操作人员设定 |
| 收货人 | 收货环节自动填充的收货操作人员 |
| 验收人 | 验收环节自动填充的验收操作人员 |
| 出库拣货人 | 出库拣货环节自动填充的操作人员 |
| 出库复核人 | 出库复核环节自动填充的复核人员 |

## 业务规则

### 新增配置

1. **职责分离强制校验**（核心业务规则）：
   - **收货人 ≠ 验收人**：否则报错"收货人和验收人不能是同一个人"
   - **出库复核人 ≠ 拣货人**：否则报错"出库人和复核人不能为同一个人"
2. **唯一性约束**：同一公司 + 同一仓库只能配置一套自动单据人员，已存在则报错
3. 自动生成配置标识

### 修改配置

- 同样强制校验收货人 ≠ 验收人、出库复核人 ≠ 拣货人

### 删除配置

- 批量删除，无额外校验

### Excel 导入配置

- 支持批量导入配置
- 失败时生成错误报告可下载

## 业务流程

### 配置使用场景

```
系统自动生成单据时
  │
  ├── 根据当前公司 + 仓库查询自动单据人员配置
  │
  ├── 收货环节 → 自动填充收货人
  │
  ├── 验收环节 → 自动填充验收人
  │
  ├── 出库拣货 → 自动填充拣货人
  │
  └── 出库复核 → 自动填充复核人
```

## 特殊场景处理

1. **职责分离合规**：GSP（药品经营质量管理规范）要求收货与验收、拣货与复核必须由不同人员执行，系统通过强制校验确保合规
2. **离职人员处理**：列表中以红色文字标示已离职人员，提醒管理员及时更新配置；选人时离职用户不可选中
3. **双模式人员输入**：部分角色（收货人、验收人等）支持手动输入模式，用于配置非系统用户的外部人员

## 操作指引

### 配置列表

**查询条件**：公司、所属仓库（级联公司）、备注。

**操作按钮**：
- **新增**：打开新建配置表单
- **编辑**：需选中一条记录
- **删除**：需选中记录
- **导入**：打开 Excel 导入弹窗

**列表显示**：

列表中每个人员字段附带离职状态检测，当人员已离职时文字显示为红色，方便管理员快速发现需要更新的配置。

人员字段包括：入库申请人、入库收货人、入库验收人、入库制单人、出库申请人、出库制单人、出库拣货人、出库复核人。

### 新建配置表单

**必填字段**：公司、仓库、入库申请人、入库收货人、入库验收人、入库制单人、出库申请人、出库制单人、出库拣货人、出库复核人。

**人员选择模式**：
- 入库申请人和出库申请人始终为远程搜索选择模式
- 其余 6 个人员字段支持**选择模式/输入模式**切换：
  - 选择模式：从用户中心搜索选择人员
  - 输入模式：手动输入，用于配置非系统用户的外部人员

**选人规则**：
- 搜索结果显示为"显示名[账号]"格式
- 离职用户为禁用状态，不可选中
- 选中用户后自动填充账号、显示名、用户 ID 等多个字段

### 编辑配置表单

- 公司和仓库不可修改
- 入库申请人和出库申请人始终为远程搜索模式
- 其余字段同样支持选择/输入模式切换

### Excel 导入

支持下载导入模板、上传 Excel 批量导入、下载错误报告。
