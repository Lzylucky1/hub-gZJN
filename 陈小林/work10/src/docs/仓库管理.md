# 仓库管理

## 业务定位

仓库是库存管理系统的顶层空间容器，所有库存操作均基于仓库展开。仓库管理负责维护仓库的基本信息、空间属性、物流对接配置以及与公司/管理员的关联关系。

## 核心概念

| 概念 | 说明 |
|------|------|
| 仓库 (StoreHouse) | 物理存储空间，库存管理的最顶层容器 |
| 仓库类型 | 自营仓库 / 三方仓库 |
| 对接系统 | 仓库对接的物流/外部系统标识（如顺丰），仅三方仓可设置 |
| 直放仓 | 商品可跳过入库环节直接放入的仓库 |
| 库位推荐 | 入库时系统自动推荐库位的功能开关 |
| 手工盘点 | 标识该仓库是否采用手工盘点模式 |
| 生产许可证确认 | 入库时是否强制确认生产许可证（仅自营仓） |
| 动态出库 | 按批预占出库扫码支持扫码动态更换批号（仅自营仓） |
| 倒批校验 | 是否做倒批校验（仅自营仓） |
| 自有运输温度信息 | 是否自有运输温度信息（仅三方仓可编辑） |

## 业务规则

### 新增仓库

- 仓库名称在同一系统中必须唯一
- 仓库编码必须唯一，不可重复
- **三方仓库特殊规则**：
  - 强制关闭"动态出库"功能
  - 强制关闭"确认批号"功能
  - 自贮存企业名称必填

### 修改仓库

- 仓库名称不可与已有仓库重名（排除自身）
- **已有库区时的限制**：若仓库下已创建库区，则对接系统和仓库编码不可修改
- **直放仓设置限制**：将仓库设为直放仓时，需检查关联公司是否已配置其他直放仓库，若有则报错
- **地址一致性校验**：修改仓库地址时，需与所有绑定公司的备案仓库地址一致（省/市/区/详细地址全匹配），否则报错
- **停用仓库校验**：停用仓库时，需检查该仓库是否存在在途单据

### 删除仓库

- 仓库下已创建库区时不允许删除
- 删除仓库时级联删除仓库与公司的关联关系及仓库管理员

### 仓库编码自动生成

编码规则：
1. 取仓库名称前 5 个字符
2. 中文字符转为拼音首字母大写
3. 英文字母直接大写
4. 数字直接保留
5. 其他字符替换为 `-`
6. 若编码已存在，取前 4 位 + 数字后缀（1-9）递增

**示例**：仓库名"华南中心仓" → HNZXC → 若冲突则 HNZX1, HNZX2...

### 物流账号配置

密码类字段存储时自动加密，支持的物流商包括：广州松冷、中健云康、佳广物流、深圳百腾、北京金邦、映急医药冷链。

## 业务流程

### 仓库查询详情流程

```
查询仓库详情
  │
  ├── 查询仓库基础信息
  │
  └── 省市区编码 → 转换为省市区名称回填
```

### 按公司查询仓库

```
查询公司可用仓库
  │
  ├── 支持按仓库类型过滤
  ├── 支持按启用状态过滤
  └── 支持按是否直放仓过滤
```

## 特殊场景处理

1. **三方仓限制**：三方仓库只能创建"合格库区"和"不合格库区"两个库区，库位也只能对应合格/不合格两种
2. **顺丰对接标识**：当对接系统为顺丰时，与三方仓同样关闭动态出库和批号确认，系统将其类型视同"三方"管理
3. **管理员关联**：仓库可绑定管理员，通过管理员账号查询其管理的仓库列表
4. **SPD 寄售仓库查询**：支持按公司+客户查询 SPD 寄售专用仓库
5. **自有运输温度信息编辑**：仅三方仓类型支持编辑

## 操作指引

### 仓库列表

支持以下查询条件：仓库名称、仓库编码、仓库类型（自营/三方）、公司、省/市/区（三级联动）、详细地址、备注、是否推荐库位、是否启用、自有运输温度信息。

**操作按钮**：
- **新增**：打开新建仓库表单
- **编辑**：需选中一条仓库记录
- **质量编辑（物管编辑）**：需选中一条仓库记录
- **删除**：需选中记录
- **编辑自有运输温度信息**：需选中一条三方仓记录
- **导出全部公司**：无限制

**多公司展示**：当仓库关联多个公司时，公司列显示第一个公司名 + 弹出按钮，点击展示所有关联公司列表。

### 新建仓库表单

**必填字段**：仓库名称、仓库类型、详细地址、省份、城市、联系电话。

**条件显示**：
- 三方仓时额外显示：对接系统（必填）、自有运输温度信息、贮存企业名称（必填）
- 自营仓时额外显示：是否做倒批校验、动态出库、补录生产许可证

**省市区级联**：选择省份变更自动清空城市和区县，选择城市变更自动清空区县。

**地址解析**：输入详细地址后可一键解析出省/市/区编码并自动回填。

**仓库编码**：仓库名称输入后自动生成编码并回填。

### 编辑仓库

- 对接系统字段中顺丰标识为系统固定值，不允许修改
- 无编辑权限时表单整体禁用

### 仓库详情页

详情页展示基础信息及右侧 Tab 面板（公司管理、物流配置、管理员等子页面）。
